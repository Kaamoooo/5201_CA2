There is no other package needed to be installed. 
main.py is the main entry function, program starting there. By setting the value USE_TRAPEZOIDAL_RULE to False or True can use different integration methods.
include.py defines the target function and intervals to be integrated, also processor numbers and problem sizes.
trapezoidal_rule.py and simpson_rule.py are the specific integration implementations with both sequential and parallel methods.
This project is available on GitHub:https://github.com/Kaamoooo/5201_CA2
